# Carbon Emissions Analysis

## 📌 Overview

This project analyzes global carbon emissions data to uncover trends, identify key contributors, and provide actionable insights through interactive dashboards. The analysis focuses on data cleaning, exploration, KPI tracking, and visualization.

## 🎯 Objectives

* Analyze carbon emissions across countries and regions.
* Identify emission trends over time.
* Compare countries based on emission levels.
* Track key environmental KPIs.
* Build an interactive dashboard for decision-making.

## 🛠️ Tools & Technologies

* Microsoft Excel
* Power Query
* Pivot Tables
* Power Pivot
* Power BI

## 📊 Dashboard Features

* Total Carbon Emissions KPI
* Country-wise Emissions Comparison
* Emissions Trend Analysis
* Top & Bottom Emitting Countries
* Interactive Filters (Slicers)
* Dynamic Charts and Visualizations
